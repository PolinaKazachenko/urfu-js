Caesar

В файле textforcaesar.txt написан текст. Программа с помощью шифра Цезаря кодирует этот код, при этом знаки препинания и другие специальные символы остаются прежними. Закодированный текст записывается в файл codedTextForCaesar.txt. Далее закодированный текст так же с помощью сдвига расшифровывается, результат записывается в файл decodedTextForCaesar.txt.
