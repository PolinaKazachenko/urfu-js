В командную строку вводится строка вида: node rle.js (команда) (имя файла1) (имя файла2)
Имя файла1 - файл для чтения
Имя файла2 - файл для записи результата
Список команд:
code - закодировать текст с помощью алгоритма RLE
decode - раскодировать строку, закодированную RLE
compare - если до этого кодировали и раскодировали текст с помощью этой программы, то можно использовать эту команду для сравнения исходного текста (для кодирования) и текста после раскодирования

Примеры ввода:
node rle.js code input.txt output.txt
node rle.js decode output.txt decode.txt
node rle.js compare input.txt decode.txt
